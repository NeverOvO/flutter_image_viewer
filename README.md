# flutter_image_viewer

Flutter基础代码实现的仿手机QQ图片放大查看器组件

## 主要功能

- 图片放大查看
- 图片上下拖动关闭
- 多图片数组滑动翻页查看
- 随意添加相关功能，例如下载图片、分享图片等
